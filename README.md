# PetCare-Management-System

##  Description

This project is a full-stack web application built using ASP.NET Core and Entity Framework.

It is designed for veterinary clinics or pet owners who manage multiple pets.

The system allows users to:
- Add a new pet to the system
- View detailed information about each pet
- Update pet information when needed
- Remove a pet or mark it as inactive

The application uses a database to store all pet data and provides a simple API to manage these records efficiently.
